每个 AI 工程师都有实时状态指示：

- ⚪ **已释放（Evicted）**：资源已回收，可重新激活

## 技术架构（给技术决策者）

### 核心技术栈

- **前端**：Next.js 14 + React 18 + TypeScript
- **样式**：Tailwind CSS + Framer Motion（2.5D 等距设计）
- **状态管理**：Zustand（轻量级、高性能）
- **数据库**：Supabase PostgreSQL + RLS 安全策略
- **实时通信**：WebSocket + Supabase Realtime
- **AI 引擎**：Anthropic Claude（最强推理能力）
- **部署**：Vercel Edge Network（全球 CDN）

### 安全架构

```
四层权限控制
├── 认证层：Supabase Auth（Magic Link + Passkey）
├── 订阅层：Construction Permit（付费验证）
├── 数据层：Row-Level Security（行级安全）
└── API 层：服务端二次验证
```

### 扩展性

- **Serverless 架构**：自动扩缩容
- **WebSocket 网关**：支持多设备实时同步
- **Pod 资源池**：EC2 按需分配

## 常见问题

### Q: AI 写的代码质量如何？

A: 每行代码都经过 QA 工程师自动审查，有完整的质量门禁机制。代码遵循业界最佳实践，包含完整的测试覆盖。

### Q: 能处理复杂的企业级项目吗？

A: 可以。系统支持导入现有 Git 项目，AI 团队可以理解现有架构并在此基础上迭代开发。

### Q: 生成的代码归谁所有？

A: 100% 归属于你。你可以自由使用、修改、商业化。

### Q: 支持什么技术栈？

A: AI 工程师精通主流技术栈，包括但不限于：React/Vue/Angular、Node.js/Python/Go、PostgreSQL/MongoDB、AWS/GCP/Azure。

### Q: 数据安全如何保障？

A: 采用银行级安全标准：传输加密、行级数据隔离、定期安全审计。你的代码只有你能访问。
