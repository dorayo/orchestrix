# Orchestrix Guide for Claude Code

For the complete workflow, see the [orchestrix Workflow Guide](../orchestrix-workflow-guide.md).

## Installation

When running `npx orchestrix install`, select **Claude Code** as your IDE. This creates:

- `.orchestrix-core/` folder with all agents
- `.claude/commands/orchestrix` folder with agent command files (`.md`)

## Using orchestrix Agents in Claude Code

Type `/agent-name` in your chat to activate an agent.

## Tips for Claude Code Users

- Commands are auto-suggested as you type `/`
