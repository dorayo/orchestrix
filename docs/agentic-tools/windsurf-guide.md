# Orchestrix Guide for Windsurf

For the complete workflow, see the [orchestrix Workflow Guide](../orchestrix-workflow-guide.md).

## Installation

When running `npx orchestrix install`, select **Windsurf** as your IDE. This creates:

- `.orchestrix-core/` folder with all agents
- `.windsurf/rules/` folder with agent rule files (`.md`)

## Using orchestrix Agents in Windsurf

Type `@agent-name` in chat to activate an agent.
