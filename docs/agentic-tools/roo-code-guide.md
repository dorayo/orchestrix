# Orchestrix Guide for Roo Code

For the complete workflow, see the [orchestrix Workflow Guide](../orchestrix-workflow-guide.md).

## Installation

When running `npx orchestrix install`, select **Roo Code** as your IDE. This creates:

- `.orchestrix-core/` folder with all agents
- `.roomodes` file in project root with custom modes

## Roo Code-Specific Features

- **Mode file**: `.roomodes` in project root
- **Mode switching**: Use mode selecto
