# Orchestrix Guide for Trae

For the complete workflow, see the [orchestrix Workflow Guide](../orchestrix-workflow-guide.md).

## Installation

When running `npx orchestrix install`, select **Trae** as your IDE. This creates:

- `.orchestrix-core/` folder with all agents
- `.trae/rules/` folder with agent rule files (`.md`)

## Using orchestrix Agents in Trae

Type `@agent-name` in chat to activate an agent.
